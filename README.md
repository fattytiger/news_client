### PC端练习项目


+ 使用Angualr框架搭建的新闻客户端
+ 数据源是从新浪网上找的(存储在本地)
+ UI框架使用的是bootstrap


### 功能列表
+ 新闻客户端首页
+ 我的主页
+ 新闻搜索页
+ 频道管理页
+ 新闻详情页
